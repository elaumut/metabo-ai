Support METABO

METABO est une application de bien-être qui analyse les données Apple Santé pour aider à comprendre les signaux physiologiques du quotidien : sommeil, récupération et activité.

Questions fréquentes

METABO a-t-il accès à mes données de santé ?

Oui, uniquement avec votre autorisation explicite via Apple Santé. METABO peut lire certaines données comme :

  • variabilité de la fréquence cardiaque (HRV)
  • fréquence cardiaque au repos
  • données de sommeil
  • activité physique

Toutes les analyses sont effectuées localement sur votre appareil.

Mes données sont-elles envoyées sur internet ?

Non. METABO adopte une approche privacy-first. Les données restent sur votre iPhone et ne sont pas envoyées vers des serveurs externes.

Comment gérer mon abonnement METABO+ ?

Les abonnements sont gérés directement par Apple via l’App Store.

Pour modifier ou annuler un abonnement :

  • ouvrez l’application Réglages
  • appuyez sur votre identifiant Apple
  • sélectionnez « Abonnements »

Comment restaurer mon abonnement ?

Dans l’application METABO, ouvrez les réglages puis utilisez l’option Restaurer les achats.

Pourquoi certaines données ne s’affichent pas ?

Certaines analyses nécessitent un historique suffisant de données Apple Santé. Si vous venez d’installer l’application, quelques jours peuvent être nécessaires avant que certaines tendances apparaissent.
